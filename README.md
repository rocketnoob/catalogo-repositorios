📱 Um site moderno e responsivo que usa a GitHub API para buscar e exibir repositórios de qualquer usuário, com filtro por linguagem de programação (JavaScript, Python, Java, etc.).

🎯 Funcionalidades

    🔍 Busca por usuário (facebook, vercel, rails, seu username!)
    🗂️ Filtro por linguagem (JavaScript, Python, TypeScript, Java, etc.)
    ⭐ Estatísticas completas (stars, forks, linguagem)
    🎨 Cores oficiais das linguagens do GitHub
    📱 100% Responsivo (mobile, tablet, desktop)
    ⚡ Carregamento rápido com loading spinner
    🔗 Links diretos para os repositórios
    🚀 Busca automática ao carregar (exibe repositórios do Facebook)

Teste com Usuários Incríveis

👑 facebook + JavaScript     → React, React Native
⚡ vercel + TypeScript       → Next.js, SWR
🐍 tensorflow + Python       → Machine Learning
☕ spring-projects + Java    → Spring Boot
🚀 gorilla + Go             → Web Framework

Sem frameworks (100% vanilla)
Sem backend (apenas GitHub API pública)
Sem build (funciona direto do navegador)

🎨 Recursos Visuais

    Gradientes modernos
    Cards com hover 3D
    Animações suaves
    Loading spinner
    Tipografia Google Fonts
    Cores oficiais GitHub
